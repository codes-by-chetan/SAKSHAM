# Mahila Bachatgat

A full-stack app for expense and savings management for a Mahila Bachatgat group.

## Project structure

- backend: Node.js + Express + MongoDB API
- frontend: React Native Expo mobile app

## Backend

1. Open the backend folder
2. Make sure MongoDB is running locally
3. Install dependencies:
   - npm install
4. Start the server:
   - npm run dev

## Frontend

1. Open the frontend folder
2. Install dependencies:
   - npm install
3. Start Expo:
   - npm start

## API overview

- POST /api/auth/register
- POST /api/auth/login
- GET /api/auth/me
- POST /api/groups
- GET /api/groups
- POST /api/expenses
- GET /api/expenses
